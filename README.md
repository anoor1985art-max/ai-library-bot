# 📚🤖 بوت المكتبة والذكاء الاصطناعي الشامل (AI & Library Assistant Bot)

بوت تليجرام متطور يجمع بين **محركات البحث العالمية عن الكتب والمراجع (`Google Books + Open Library`)** وبين **قدرات الذكاء الاصطناعي الفائقة (`LLM AI Assistant & Translator`)**.

---

## ✨ مميزات البوت الرئيسية:
1. 📚 **البحث في المكتبة العالمية:**
   - ابحث عن أي كتاب أو رواية أو مؤلف عربي أو أجنبي.
   - يعرض لك البوت فوراً: (اسم الكتاب، المؤلف، سنة النشر، التقييم، عدد الصفحات، والبوستر الرسمي).
   - أزرار تفاعلية للبحث عن روابط تحميل `PDF` المجانية أو القراءة المباشرة عبر `Google Books`.
2. 🧠 **التلخيص والتحليل الذكي للكتاب:**
   - بضغطة زر واحدة على أي كتاب، يقوم الذكاء الاصطناعي بقراءة بياناته واستخراج **أهم 5 أفكار رئيسية وسبب أهمية قراءة الكتاب** بلغة عربية فصحى راقية.
3. 🤖 **المحادثة الذكية والإجابة على الأسئلة:**
   - اسأل المساعد الذكي عن أي موضوع في العلوم، البرمجة، الفلسفة، أو الأدب.
4. 🌐 **المترجم الذكي الاحترافي:**
   - ترجمة أدبية وعلمية دقيقة بين العربية والإنجليزية وباقي لغات العالم.
5. ✍️ **تلخيص النصوص والمقالات:**
   - أرسل أي نص طويل أو مقال وسيقوم الذكاء الاصطناعي بتفكيكه وتلخيصه إلى نقاط منظمة وموجزة.

---

## 🛠️ طريقة التشغيل والرفع على Render:

### 1️⃣ إعداد توكن البوت (`TELEGRAM_BOT_TOKEN`):
1. اذهب في تليجرام إلى `@BotFather` وأنشئ بوتاً جديداً عبر الأمر `/newbot`.
2. خذ التوكن الخاص بالبوت وضعه في ملف `.env` داخل المشروع بهذا الشكل:
   ```env
   TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
   ```
   *(يمكنك أيضاً إضافة `GEMINI_API_KEY` أو `OPENAI_API_KEY` إن رغبت، وفي حال عدم وضعها سيعتمد البوت تلقائياً وبشكل مجاني 100% على محركات `Pollinations AI` السريعة).*

### 2️⃣ التشغيل المحلي على جهازك:
```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python ai_library_bot.py
```

### 3️⃣ الرفع على خادم سحابي (Render.com) للعمل 24/7:
1. ارفع هذا المجلد إلى حسابك على GitHub كـ Repository جديد باسم `ai-library-bot`.
2. في موقع `Render.com` اختر **`New -> Web Service`** واربطه بالمستودع.
3. في خانة **Start Command** اكتب:
   ```bash
   gunicorn ai_library_bot:app --bind 0.0.0.0:$PORT
   ```
4. في خانة **Environment Variables** أضف المتغير:
   - `TELEGRAM_BOT_TOKEN`: (توكن البوت الخاص بك).
5. بعد اكتمال البناء، خذ رابط موقع Render وأضفه في موقع `cron-job.org` بزيارة كل 5 دقائق لحمايته من السكون!
